# BeaverDeck 1.3.2

`BeaverDeck 1.3.2` is a UI polish and pods filtering release focused on faster triage and denser day-to-day operations workflows.

## Highlights

- added pod filtering by name regex and status
- scoped pod batch selection and batch actions to the filtered result set
- made navigation and action controls denser across the UI to reduce wasted space

## Added

- pod filtering by:
  - name regex
  - status
- validation feedback for invalid pod name regex filters

## Changed

- pod batch select-all now applies only to the currently filtered pods
- batch `Evict` and `Delete` actions now operate only on the currently filtered selection
- removed the redundant pod namespace filter from the Pods toolbar because namespace preselection already exists elsewhere in the UI
- reduced spacing and font sizes in the sidebar navigation to make the menu denser
- reduced general button sizing across the UI so controls are no taller than the navigation buttons where practical

## Fixed

- fixed a safety issue where filtered pod batch operations could otherwise imply broader scope than the current filtered view

## Deployment Notes

- app version: `1.3.2`
- Helm chart version: `2.0.2`
- no chart breaking changes in this release
